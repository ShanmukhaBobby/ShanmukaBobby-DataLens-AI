# 🔍 DataLens AI — Automated Database Metadata Intelligence

Upload any SQL database → AI maps every table, column and domain instantly.

## What it does
- Reads any SQL file with 1000+ tables
- AI generates descriptions for every table and column
- Outputs a professional 3-sheet Excel report
- Complexity scoring per table
- Business domain detection

## Built with
- Python, Groq AI, LLaMA 3.3
- Google Colab
- OpenPyXL
